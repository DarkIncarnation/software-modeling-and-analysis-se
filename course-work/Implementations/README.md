Описание:
Този проект представлява пълен модел на базата данни на търговската уеб платформа Amazon. Проектът включва: концептуална диаграма, физическа диаграма на Data Warehouse (3 диаграми), както и Power BI доклад.

Използвани инструменти:

   -draw.io

   -MS SQL Server

   -SQL Server Management Studio

   -Power BI Desktop

Инструкции за създаване на базата данни:

1. Необходимо е инсталирането на MS SQL Server и SQL Server Management Studio.
2. След отваряне на SQL Server Management Studio направете връзка със сървъра.
3. Отворете и изпълнете файловете с разширение .sql в следната последователност:
   database.sql > functions.sql > stored_procedures.sql > triggers.sql
4. Ако стъпките са правилно изпълнени, ще бъдат създадени базата данни, функции, съхранени процедури и тригери.
5. Отворете и изпълнете останалите файлове с разширение .sql в следната последователност:
   Insert_1.sql > Insert_2.sql > Insert_3.sql
6. Ако стъпките са правилно изпълнени, базата данни ще бъде напълнена със записи.
